Basic stuff for making a webpage

<!doctype html> <html> <head> <meta charset="utf-8"> <title></title> </head> <body> </body> </html>

What this does

<!doctype html> - This tells the computer which version of HTML this program uses (HTML5). <html> - This tells the computer that there is HTML programing here. <head> - This begins the head section, where you specify things about the page. <meta charset="utf-8"> - This tells the computer which set of symbols it understands (US, Chinese, ect.). <title></title> - This is where you would put the name of you're page </head> - This ends the head section <body> - This begins the body section, where you put the coding for the actual webpage. </body> - This endsthe body section </html> - This ends the HTML code To start something, put it inbetween two angled brackets, like this: <_____> To end something, put a forward slash after the first angled bracket, like this: </_____>

Tags

There are different types of tags depending on what you want to do. Here are some examples:

Headings

There are six different levels of headings:

  • <h1> - This is a tag for the main heading.
  • <h2> - This is a tag for a sub-heading
  • <h3> - This is a tag for a sub-sub-heading
  • <h4> - This is a tag for a sub-sub-sub-heading
  • <h5> - This is a tag for a sub-sub-sub-sub-heading
  • <h6> - This is a tag for a sub-sub-sub-sub-sub-heading

Lists

There are two different types of lists:

  • <ul> - Unordered list (bullets)
  • <ol> - Ordered list (Numbers)
  • <li> - List item

Emphasis

  • <em> - Emphasis (italics)
  • <strong> - Strong emphasis (bold)

Other

  • <p> - paragraph
  • <br> - Line Break (enter)
  • <xmp> - What's inside is text, not programing
